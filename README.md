### Why Spotter?
* Spotter is a place where users can their online identity to stay connected with each other.

***

<a name="features"/>

### What are the features ?
* Authentication
  * Signup
  * Login
  * Logout
* Filter posts by
  * Trending
  * Latest
  * Oldest
* Create a post.
  * Upload images
  * Like
  * Comments
* Edit & delete post
* User feed 
* User profile
  * Add profile picture
  * Bio
  * Portfolio
* Follow/Unfollow user
* Explore feed``
* Add/Remove to/from bookmark
***

### What about tech stack?
